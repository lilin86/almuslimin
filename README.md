
# AlMuslimin: Mahalil Aisha Guidance

A comprehensive educational platform for learning Quran, Tauhid, Fiqh, and Hadith, designed for both Muslims and seekers of knowledge.

## Features
- **Learning Modules**: In-depth guidance on Quran, Tauhid, Fiqh, and Hadith.
- **AlMuslimin Alim AI**: A compassionate AI assistant for spiritual and technical questions.
- **Practical Tools**: Digital Tasbih counter and Zakat calculator.
- **Knowledge Quizzes**: AI-generated quizzes to test your learning.
- **Way of Life**: Ethics, finance, and mental health through an Islamic lens.

## Deployment
This app is optimized for Vercel. 

### Environment Variables
Ensure you set the `API_KEY` in your Vercel project settings to your Google Gemini API key.
